== Description ==

A Simple Wordpress Plugin That Removes The Version and Other Details That Is under Each Plugin.

View the screenshots for more details on how it looks.

Note
There are no settings.

== Installation ==


1. Go to the Plugins> Add New> Upload in you admin dashboard.

2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.
